# logicproject
 Logic project dos xarás

# Tutorial
 Para rodar, abra a pasta functionalscript/ no seu terminal e escreva   bash zedsolver.sh   (ele fará os dois códigos rodarem gerando suas respectivas saídas)

# Ps:
 Se precisar trocar o arquivo Entrada.in para testar outra entrada, NÃO substitua o arquivo que já está lá (copiando e colando o arquivo). Ao invés disso, altere a entrada via editor de texto (copiando do outro arquivo e colando nele).
O motivo é um bug que, caso copie e cole na pasta o arquivo da entrada, por razões desconhecidas a saída Resolucao.out sairá errada (pode testar se quiser).
Caso tenha (acidentalmente ou não) deletado o arquivo Entrada.in do diretório onde está rodando o código, não copie o arquivo novamente, é necessário copiar o texto da entrada em um novo arquivo Entrada.in criado dentro do diretório.

# Ps2:
 Caso descubra o motivo do bug, por favor informar aos criadores.
